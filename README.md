# 课程介绍

在本课程中，您将学习使用 Jetpack Compose 构建 Android 应用的基础知识。

Jetpack Compose 是用于构建 Android 应用的全新界面工具包。

在这个过程中，您将开发一些应用，为成为一名 Android 开发者奠定基础。
